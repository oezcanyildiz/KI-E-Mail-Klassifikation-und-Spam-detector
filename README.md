##KI-gestützte E-Mail-Klassifikation

Prototyp zur automatisierten Spam-Erkennung und Themenzuordnung

Thema des Projekts

Dieses Projekt entsteht im Rahmen meines IHK-Abschlussprojekts 2025/2026.
Ziel ist die Entwicklung eines Prototyps, der eingehende E-Mails automatisch bewertet, zunächst auf Spam prüft und anschließend legitime Nachrichten in inhaltliche Kategorien einordnet (z. B. Mitarbeiterangelegenheiten, Zahlungen, Support). Auf Basis des Ergebnisses wird eine Weiterleitungsentscheidung simuliert.

#Projektbeschreibung

Der Prototyp soll:
  
  E-Mails einlesen und vorverarbeiten
  
  eine Spam-Erkennung durchführen
  
  legitime E-Mails thematisch klassifizieren
  
  die Modellgüte evaluieren (Accuracy, F1-Score usw.)
  
  verschiedene Modellansätze vergleichen
